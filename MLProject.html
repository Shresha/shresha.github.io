Song recommendation and genre prediction using Machine Learning

As a team of 5, we explored song genre prediction, clustering and recommendation based on the features of a song as part of the Machine Learning course at Georgia Tech. Previously, there has been research in the fields of song clustering using K-Means and Hierarchical clustering, as well as recommendation of songs on spotify. In this project we took a mixed approach, where we clustered the data using different unsupervised learning methods, and then applied supervised learning while taking into account other factors to predict genres of a song over multiple models. Moreover, we compared the algorithms, visualized and analyze them here. We leveraged the Spotify API to test our models and got a genre prediction given a song URL. To further explore our project, you can visit: https://github.gatech.edu/pages/ysinghal6/ML-Project/final
